Information

  • This page uses OpenPGP.js to encrypt text/files.
  • This encryption process takes place entirely in your browser.
    • No information entered here is sent to any servers.
  • For more information, please visit this GitHub repository: https://github.com/thisaaronm/OpenPGP-Web-App

Select Recipient

The selected user's public key will be used for encryption.

Encrypt Text

  • In the text area below, enter the text you wish to encrypt.
  • Click the Encrypt button.
  • The entered cleartext will be replaced with the encrypted text, and a Copy button will appear.
  • Clicking the Copy button will copy the encrypted text to your clipboard.

Text box content has been copied to the clipboard.

Encrypt Files

  • Click the button below to select the file(s) that you wish to encrypt.
    • There is a 5 MB (5000 KB) file size limit per file.
    • Files that meet this size limit will appear in the Included Files section.
    • Files that do not meet this size limit will appear in the Excluded Files section.
  • The Included Files list will contain downloadable links with .gpg appended to the file name.
    • Your original files are NOT modified.
    • Only encrypted copies of your files are created.
  • Click each link to save the encrypted file to your local computer.
